VFD-Library-HS321
=================
API for controlling the frequency converter HS321 via the Modbus RTU protocol.

Imples the interaction with the frequency via the RS485 interface.

Supports reading/writing parameters, sending control commands and diagnostics.

The VFD HS320 & 321 Series is described in more detail [here](http://www.hs-bp.com/enproducts_show.aspx?id=7)
